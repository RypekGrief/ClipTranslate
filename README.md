# ClipTranslate
### Açıklama
**ClipTranslate**, Windows içerisinde çalışan, seçilen metini CTRL + SHIFT + X tuşlarana basarak İngilizce'ye çeviren bir çeviri uygulamasıdır.

## Gereksinimler
- [Python](https://www.python.org/downloads/windows/)
- requirements.txt dosyasında ki tüm kütüphanaler indirilmesi gerekir. (Klasör içerisinde CMD açarak `py -m pip install -r requirements.txt` yazın)
- [AutoHotkey V2](https://www.autohotkey.com/?__cf_chl_f_tk=A5jnZlgXkCamN5B5fiZDsWd2ML7sYOrbAfyCfAjUnbc-1783006160-1.0.1.1-6TIx.Ag1vvTq.ZsrXgb5_BZiy3W57c36MhiIdCz.I_A)

## Özellikler
- Herhangi bir dil İngilizce'ye çevrilebilir.
- Temel olarak yazı yazılıp, metin seçilebilen her yerde program çalışır. (Oyunlar, Tarayıcılar, Not Defteri, VS Code vb.)
- Eğer sistem tepsisinde `Windows ile Başlat` seçeneğini işaretleyerek her sistem başlangıcında program otomatik olarak açılır.
- İsteğe bağlı olarak çeviri yapıldıktan sonra bildirim gösterilir.
- Program kapatılmadan sistem tepsisi üzerinden `Aktif` butonu kapatılarak çeviri uygulaması durdurulabilir.
- RAM kullanımı fazla değildir. (0-50 MB)

## Kullanım
1. Programı başlatın, eğer bilgisayarınızda AutoHotkey kurulu değilse uyarı verir.
2. Yazı yazılıp, metin seçilebilen herhangi bir yere istediğiniz dilde bir şey yazın. (İngilizce olmayan)
3. CTRL + A tuşlarına basarak veya imleç ile yazıları seçin.
4. CTRL + SHIFT + X tuşlarına basın, bildirimler kısmında `Metin Çevrildi ve Panoya Kopyalandı` gözükecek. (Sistem tepsisinden kapatılabilir)
5. CTRL + V'ye basın.

Çeviri uygulanmış olacaktır.
