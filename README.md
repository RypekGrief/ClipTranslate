# ClipTranslate
### Açıklama
**ClipTranslate**, Windows içerisinde çalışan, seçilen metini CTRL + SHIFT + X tuşlarana basarak İngilizce'ye çeviren bir çeviri uygulamasıdır.

## Gereksinimler
- [Python](https://www.python.org/downloads/windows/)
- requirements.txt dosyasında ki tüm kütüphanelerin indirilmesi gerekir. (Klasör içerisinde CMD açarak `py -m pip install -r requirements.txt` yazın)
- [AutoHotkey V2](https://www.autohotkey.com/?__cf_chl_f_tk=A5jnZlgXkCamN5B5fiZDsWd2ML7sYOrbAfyCfAjUnbc-1783006160-1.0.1.1-6TIx.Ag1vvTq.ZsrXgb5_BZiy3W57c36MhiIdCz.I_A)

## Özellikler

- Seçili metni anında İngilizceye çevirir.
- Metin seçilebilen ve kopyalanabilen neredeyse tüm uygulamalarda çalışır (oyunlar, tarayıcılar, Discord, Not Defteri, Visual Studio Code vb.).
- Sistem tepsisinden **Windows ile Başlat** seçeneği etkinleştirilerek bilgisayar açıldığında otomatik olarak başlatılabilir.
- Çeviri tamamlandığında isteğe bağlı olarak masaüstü bildirimi gösterir.
- Uygulamayı kapatmadan sistem tepsisindeki **Aktif** seçeneği ile çeviri özelliği açılıp kapatılabilir.
- Hafif bir uygulamadır ve genellikle **50 MB'den daha az RAM** kullanır.

## Kullanım
1. Uygulamayı başlatın, eğer bilgisayarınızda [AutoHotkey](https://www.autohotkey.com/?__cf_chl_f_tk=A5jnZlgXkCamN5B5fiZDsWd2ML7sYOrbAfyCfAjUnbc-1783006160-1.0.1.1-6TIx.Ag1vvTq.ZsrXgb5_BZiy3W57c36MhiIdCz.I_A) kurulu değilse uyarı verir.
2. Yazı yazılıp, metin seçilebilen herhangi bir yere istediğiniz dilde bir şey yazın. (İngilizce olmayan)
3. CTRL + A tuşlarına basarak veya imleç ile yazıları seçin.
4. CTRL + SHIFT + X tuşlarına basın, bildirimler kısmında **Metin Çevrildi ve Panoya Kopyalandı** gözükecek. (Sistem tepsisinden kapatılabilir)
5. CTRL + V'ye basın.

Çeviri uygulanmış olacaktır.
